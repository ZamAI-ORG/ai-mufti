# AI Mufti Roadmap

Current Status: Planning & Initial Development

## Now (Q2 2026)
- [ ] **RAG Pipeline:** Index core texts (Quran, Sahih Bukhari, Sahih Muslim) for retrieval.
- [ ] **Citation Engine:** Ensure every answer includes precise source references (Book/Volume/Hadith #).
- [ ] **Basic Q&A Interface:** Clean chat interface for asking religious questions.
- [ ] **Search Functionality:** Keyword search across the indexed knowledge base.

## Next (Q3 2026)
- [ ] **Context Awareness:** Handling follow-up questions and maintaining conversation context.
- [ ] **Multi-language Support:** Seamless switching between Arabic, English, and Pashto.
- [ ] **Source Verification:** Links to original scanned pages of source texts where possible.
- [ ] **Topic Explorers:** Curated guides on common topics (Prayer, Fasting, Zakat).

## Later (2027+)
- [ ] **Scholar Verification Portal:** Interface for qualified scholars to review and verify AI answers.
- [ ] **Voice Interface:** Voice-to-voice interaction for accessibility.
- [ ] **API Access:** Developer API for integrating Islamic knowledge into other apps.
- [ ] **Fiqh Settings:** Ability to toggle between different schools of thought (Madhabs).
